# Notas:

Este es un programa para multiplicar:

```
Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Manda el número base de la tabla de multiplicar
                                                             [number] [required]
  -l, --listar   Muestra la tabla en consola          [boolean] [default: false]
  -h, --hasta    El límite de la tabla                  [number] [default: "10"]
```
